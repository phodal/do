一步步设计 DevOps

入门指南

  1. 了解 DevOps 所需的技能。通过 『技能图谱』了解相关的知识。
  2. 完善所需的 DevOps 知识。结合 『智库』 完善
  3. 了解 DevOps 如何实施。通过 『操作手册』 提供的模式进行设计
  4. 寻找好的 『DevOps 实践』。

实施指南

  1. 可视化现有的 DevOps 流程。通过 『设计 DevOps』进行设计
  2. 重新设计新的 DevOps 流程。结合 『案例学习』中的一系列示例完善
  3. 在 DevOps 流程中引入对应的工具。使用 『DevOps 元素周期表』介绍的一系列工具
  4. 完善关键节点中的设计。结合 『检查清单』 中的检查清单完善

DevOps 平台开发指南（TBD）

  1. 了解 DevOps 平台如何设计。 -> 『DevOps 平台实践』
  2. 了解市面上现有的 DevOps 解决方案。 -> 『解决方案』
